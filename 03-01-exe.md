#### 3-5 文字の装飾

  1. 文字のサイズ
  
  ```css
    html {
        font-size: 100%;
    }

    h1 {
        font-size: 3rem;
    }

    h2 {
        font-size:20px;
    }  
  ```

  2. フォントの種類

  ```css
    html {
        font-size: 100%;
        font-family: serif; /*追加*/
    }

    h1 {
        font-size: 3rem;
        font-family: YuGothic; /*追加*/
    }

    h2 {
        font-size:20px;
        font-family:"ヒラギノ丸ゴ Pro W4"; /*追加*/
    }  
  ```

  3. 文章をそろえる

  ```css
 　/*h2の下に追加*/
    p{
        text-align: center;
    }
  ```

#### 3-6 Webフォントを使う

  1. Google Fonts(https://fonts.google.com/)
  - サイトからリンクを取得
  - HTMLのheadタグ内にリンクを追加
  - CSSにプロパティーを追加

#### 3-7 色をつける

  ```css
    html {
        font-size: 100%;
        font-family: serif;
        background-color: #fee; /*追加*/
    }

    h1 {
        font-size: 3rem;
        font-family: YuGothic;
        color: #0bd; /*追加*/
        background-color: rgb(238, 254, 255); /*追加*/
    }

    h2 {
        font-size:20px;
        font-family:"ヒラギノ丸ゴ Pro W4";
        color: navy; /*追加*/
    }
    p{
        text-align: center;
        color:rgba(10,130,165,.5); /*追加*/
    }  
  ```

#### 3-8 背景画像
  
  1. 画像ファイルを「images」フォルダにコピー
  - (program/images/bg.png)
  - (program/images/bg-airplane.png)
  (https://github.com/webgakushu2020/webcorse/tree/main/program/images)

```css
    html {
        font-size: 100%;
        font-family: serif;
        background-color: #fee;
    }

    /* ここに追加↓ */
    body {
        background-color: #fee;
        background-image: url(../images/bg.png); /*追加*/
    }
    /* ここに追加↑ */

    h1 {
        font-size: 3rem;
        font-family: YuGothic;
        color: #0bd;
        background-color: rgb(238, 254, 255);
    }

    h2 {
        font-size:20px;
        font-family:"ヒラギノ丸ゴ Pro W4";
        color: navy;
    }
    p{
        text-align: center;
        color:rgba(10,130,165,.5);
    }  
  ```

  2. 背景画像の大きさを指定
   
   ```css
    html {
        font-size: 100%;
        font-family: serif;
        background-color: #fee;
    }

    body {
        background-color: #fee;
        /* background-image: url(../images/bg.png); */　/*コメントアウト*/
    }

    h1 {
        font-size: 3rem;
        font-family: YuGothic;
        color: #0bd;
        background-color: rgb(238, 254, 255);
        background-image: url(../images/bg.png);
    }

    h2 {
        font-size:20px;
        font-family:"ヒラギノ丸ゴ Pro W4";
        color: navy;
    }
    p{
        text-align: center;
        color:rgba(10,130,165,.5);
    }  
  ```
